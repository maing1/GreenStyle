# GreenStyle 
GreenStyle is a responsive sustainable fashion website built using HTML, CSS, JavaScript, Bootstrap 5, and Google Fonts. The website showcases eco-friendly fashion products and includes interactive features such as a shopping cart, wishlist, image gallery, contact form validation, and a checkout page.

## Features
- Responsive Bootstrap navigation bar
- Home page with hero section and sustainability highlights
- About page introducing the GreenStyle brand
- Gallery page with image gallery and slideshow
- Collection page displaying products
- Add to Cart functionality
- Wishlist functionality
- Checkout page displaying cart items
- Clear Cart feature
- Contact form with JavaScript validation
- Responsive design for desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Google Fonts (Fraunces & Inter)
- Local Storage (for saving cart items)

## Pages
### Home
Introduces GreenStyle and highlights the company's sustainability goals.

### About
Provides information about GreenStyle's mission and values.

### Gallery
Displays product images and a simple slideshow.

### Collection
Displays products that users can:
- Add to Cart
- Add to Wishlist

### Checkout
Shows all items added to the shopping cart and allows users to clear the cart.

### Contact
Contains a contact form with JavaScript validation.


## JavaScript Features
### Contact Form Validation
- Checks that all required fields are completed.
- Displays success and error messages.

### Shopping Cart
- Add products to the cart.
- Cart count updates automatically.
- Cart items are stored using Local Storage.
- Cart items are displayed on the Checkout page.
- Clear Cart removes all items.

### Wishlist
- Allows users to save favourite products.
- Displays a confirmation message when an item is added.


## How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (recommended).
4. Right-click **index.html** and select **Open with Live Server**.
5. Navigate through the website using the navigation bar.
